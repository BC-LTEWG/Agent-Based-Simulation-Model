# BC-LTEWG Development Schedule

- 2025-11-16 — Purchases and transfers to send *pull* signals: 
  - Person makes purchases at appropriate frequencies.
  - Distributor exchanges goods for hours on purchases.
  - Distributor chooses a Producer from whom to order a product in order to maintain the inventory.
  - Producer signals when an order has been accepted (with a plan) and when it is complete.
  - If no producer produces the product, the distributor must source it from the capitalist economy — or a producer must tool up to produce it.  (Represent this as a black box until later. The decision logic starts in the Distributor.)
    - Adapt \`Alex’s model with dependency map (Zhejun). 
- 2025-11-23 — Production:
  - Implement prices, price computations, and price adjustments.
    - `Product` should have two API functions: `get_costs` and `get_price`.  The former returns a data structure with three fields: a vector of machines, a vector of pairs of raw material and quantity per output product (where the unit is the product’s “natural unit,” the same as the unit used by its producers), and the third is a scalar integer, the necessary living labor.  The latter returns the sum per output product.  How we handle the machine component is TBD.
  - Plan approval: if PRD is not close to 0, the next plan must assume the previous plan’s hours-to-production ratio.  (Pain signal.)
  - Start to develop LTE-CE trading model based on theoretical framework (Zhejun). 
- 2025-11-30 — Worker allocation: 
  - Producer hires workers it needs (in addition to workers it has) based on required and available skills.
  - Producer transfers hours to workers.
  - Run the model for a specified time or until a product cannot be produced.
- 2025-12-7 — Public sector: 
  - Apply the FIC to workers’ pay.
  - Public revenue goes into a social fund.
- 2025-12-14 — Children and the elderly: 
  - Extend Person to represent full life span (and set working age).
  - Determine average consumption per person.
  - Draw accordingly from the social fund to children and retired people.
- 2025-1-18 — Education, training, and tooling: 
  - Set up a public workforce (a set of producers).
  - Determine the costs of general education and implement payment from the social fund.
  - Determine work-related training and draw those hours from the social fund.  (How are these hours represented in a plan and in time to delivery?)
- 2025-1-25 — Trade with the capitalist economy.
- 2025-2-1 — Clean up the code base.
- 2025-2-8 — Start drafting a paper.
