# Agent-Based Simulation

- To build the project, run `make`. To run the simulation, enter `/bin/sim/` or `cd bin ; ./sim`.
- To build unit tests, run `make tests`'. To run them, enter `cd bin ; ./run_tests.sh`. Note that you cannot run the `run_tests.sh` script from outside of `bin` because it uses a glob expression that depends on the directory from which the script is run.
